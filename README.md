# SwiftUI_DropDownMenu-ChangeOptionOnHover

A Custom Dropdown (Up) Menu that can Change Option as Hovering. Like the system Switch keyboard (Globe) button.

Specifically, with the following behavior.
- Long press to bring up the menu
- User can transform the long press to a hover (drag) directly and start selecting the option
- User can select either with dragging or tapping.
- Upon selection (gesture inside the menu), menu dismiss.


For more details, please refer to my blog [SwiftUI: Custom Dropdown (Up) Menu + Change Option as Hovering](https://medium.com/@itsuki.enjoy/swiftui-custom-dropdown-up-menu-change-option-as-hovering-d4e0b24c317d)
